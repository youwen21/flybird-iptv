# flybird-iptv
flybird-iptv 是一款iptv检查器 

## 系统支持
 - Windows (windows10, windows11)
 - MacOS

## 功能介绍
### 导入IPTV
 - 导入m3u
 - 导入txt
 - url导入

### 检查iptv是否生效
 - 批量检查
 - 配置代理，通过代理检查
 - 批量删除检查错误次数大于等于2的记录

### 导出iptv
 - 导出m3u
 - 导出txt

### 播放
 - 播放TV

### 设置
 - 启动代理
 - 软件激活
